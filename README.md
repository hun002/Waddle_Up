# Waddle_Up!

📁프로젝트 소개<br><br>
플래시 게임 '황새 오래 걷기'를 참고하여 제작한 오리 오래 걷기 게임입니다. <br>
 * 원본 플래시 게임의 작동 로직을 확인할 수 없어, 게임의 기본 컨셉을 참고하고 세부 게임 로직은 직접 설계하였습니다.

⌛개발 기간<br>
집중 개발 기간: 2025년 12월 12일 ~ 2025년 12월 24일 (12일)<br>
전체 프로젝트 기간: 2025년 4월 18일 ~ 2025년 12월 24일<br>
- 초기: 게임 기획 및 기본 로직 구성<br>
- 후반: 세부 기능 구현 및 완성도 개선<br>

👩🏻‍💻멤버 구성 및 역할분담<br>
주찬미(조장): 게임 전체 로직 설계 및 기능 구현<br>
동아리 아트 디렉터(부조장): 동아리 아트 디렉터(부조장): 게임 썸네일 및 오리 파츠 디자인<br>
동아리 아트: 게임 배경 및 세부 디자인 요소 제작<br>
  
🖥️개발 환경<br>
개발 플랫폼: Windows 11<br>
개발 툴: Visual Studio Code, Unity Hub <br>

👾기술 스택<br>
언어: C# <br>
기타: Github<br>

🔍주요 기능<br>

- 메인 타이틀<br>
게임 설명 버튼, 게임 설명 창, 시작 버튼 및 움직이는 썸네일 구현

- 오리<br>
오리 프리팹 제작 및 머리를 중심으로 한 움직임 로직 구현
* 원본 게임의 로직 코드를 확인할 수 없어 기존 몸통 기반 움직임 방식을 분석 후, 머리를 중심으로 한 새로운 움직임 로직으로 재구성하였습니다.

- 게임 오버<br>
게임 오버 화면 및 재시작 버튼, 오리 머리 충돌 시 게임 오버 처리 기능

- 배경<br>
배경 프리팹 제작 및 바닥/배경 오브젝트 무한 생성, 이전 배경 제거 기능

- 오디오<br>
게임 오버 상황에서 효과음 재생 기능
